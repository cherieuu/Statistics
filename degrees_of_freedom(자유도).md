# degrees of freedom



자유도의 개념이 가장 먼저 나오는 것이 

- unbiased variance(불편 분산)
- sample variance(표본 분산)



결론적으로,

불편분산에서는 편차의 제곱의 합 / n

표본분산에서는 편차의 제곱의 합 / n-1



ex. 만약 모집단의 데이터의 개수가 1000개이고,

이중에서 임의로 우리가 선택한 표본집단이 10개라고 하면,

불편분산 = 표본의 편차의 제곱의 합 / n

표본분산 = 표본의 편차의 제곱의 합 / n-1



--> n으로 나누고, n-1으로 나누는 차이는...?



불편분산은 우리가 보통 편차를 구하는 것처럼 표본들의 편차의 제곱의 합을 n으로 나눈다.

하지만, 모집단의 데이터 중에서 임의로 10개만을 추려서 진행하고 있다.

그러므로, 우리 원하는 모집단의 분산과 비교했을 때, 오차가 생길 수 있다.

unbiased, 바이어스 되지 않았다는 의미라고 생각하면 되겠다.



왜 오차가 생길까? 라고 생각한다면,



![sample1](C:\Users\mh353\Desktop\sample1.png)



모집단 14개의 데이터 중에 우리가 표본 3개를 취했는데, 왼쪽 3개를 취했다고 생각해보자.

이 3개의 데이터는 모여 있으니, 그냥 나누기 3을 해서 분산을 구하면,

모집단의 분산보다 작게 나오게 된다. ( 직관적인 생각 )

그래서 n-1을 함으로써, 나누기 값을 작게 해서 분산의 값을 크게 만든 것이 표본분산이다.
